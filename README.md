# Project_Lady_Slash

Developed with Unreal Engine 5.7

Based on the Udemy Course Unreal Engine 5 C++ The Ultimate Game Developer Course by Stephen Ulibarri  

Project_Lady_Slash is a third-person action RPG prototype built in Unreal Engine 5 using C++ and Blueprints.
The project focuses on gameplay programming, combat systems, reusable AI architecture, open world environment creation, and modern Unreal Engine 5 systems while following scalable game development practices.

The game was developed from an empty project and gradually expanded into a fully playable open-world hack-and-slash RPG experience featuring melee combat, enemy AI, loot systems, destruction systems, animation systems, visual effects, and exploration mechanics.
